Benchmark Construction

How TailSkills builds 208 task variants

TailSkills constructs 208 exception-heavy task variants from 54 base tasks using 14 variant types across 6 tail categories (Encoding, File System, Data Quality, Network, Dependency, Security). Each variant injects a specific tail condition that tests whether the agent's skill retains rare exception-handling knowledge.

  1. Base tasks: 54 SkillsBench tasks with deterministic verifiers and oracle solutions.
  2. Variant injection: 14 types of tail conditions (BOM encoding, read-only output, NaN poisoning, DNS jitter, missing dependencies, etc.)
  3. Oracle verification: every variant is confirmed solvable by an oracle agent before inclusion.
  4. Result: 208 verified task variants that specifically stress tail-knowledge preservation under skill distillation.
